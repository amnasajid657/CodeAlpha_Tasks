# 💳 Credit Scoring Model (Machine Learning)

This project predicts whether a customer is creditworthy using machine learning classification models.

---

## 🎯 Objective
To analyze financial data and classify customers as:
- ✔ Creditworthy (Approved)
- ❌ Not Creditworthy (Rejected)

---

## 🧠 Approach
- Data preprocessing using Pandas & NumPy
- Feature scaling using StandardScaler
- Classification using multiple ML models:
  - Logistic Regression
  - Random Forest
  - XGBoost (optional)

---

## 📊 Features Used
- Income
- Age
- Debt
- Late Payments

---

## ⚙️ Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy

---

## 🚀 How to Run
```bash
pip install pandas numpy scikit-learn xgboost
python credit_model.py
